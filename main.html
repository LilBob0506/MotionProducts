About the Project

The main goals for the project is to create a program that scrapes either the Manufacturers webpage or do a Google search to accurately find images of a part given the Manufacturer name, product name, and/or product number.

Features

Accurately finding part images

Reformating the images to be named and sized a certain way.

Potentially create a UI for determining if the images found are good to be uploaded or not

Potentially uploading all approved images onto Motion.com
